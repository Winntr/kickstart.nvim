### `lua/plugins/ui.lua`

**Added plugins:** `stevearc/aerial.nvim`, `kevinhwang91/nvim-bqf`

#### aerial.nvim (symbol outline)

| Key | Description |
| --- | ----------- |
| `<leader>lo` | Symbol outline |
| `<leader>ln` | Symbol navigation |
| `{` / `}` | Prev/next symbol (buffer-local when aerial attaches) |

Lazy loading: `cmd`, `keys`. Backends: LSP, Treesitter, Markdown, Man. Layout: min_width 30, prefer right.

#### nvim-bqf (better quickfix)

Lazy loading: `ft = "qf"`. Auto-enables with resize, preview (height 12, rounded border).

Quickfix-local keymaps: `<cr>` open, `o` openc, `<C-s>` split, `<C-v>` vsplit, `t` tab, `<C-p>`/`<C-n>` prev/next file, `<Tab>`/`<S-Tab>` toggle selection.

## Post-install

1. Open Neovim
2. Run `:Lazy sync` to install all new plugins
3. Run `:Mason` to install formatters/linters if needed (`stylua`, `ruff`, `prettierd`, `eslint_d`, `isort`, `black`)
4. For coverage: ensure test runners produce coverage reports in expected paths
5. For kulala: create `.http` files to use the REST client
